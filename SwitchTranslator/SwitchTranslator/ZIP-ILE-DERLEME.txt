ZIP ILE BASKA PC'DE DERLEME
============================

1) scripts\zip-for-build.ps1 calistir (PowerShell).
   Masaustunde "SwitchTranslator-for-build-....zip" olusur.

2) Zip'i USB ile baska bilgisayara tasiyip klasore cikar.

3) O PC'de:
   - Android Studio: File -> Open -> cikarilan SwitchTranslator klasoru
   - Gradle bitsin -> Build -> Build APK
   VEYA
   - JDK 17 + Android SDK kuruluysa: gradlew.bat assembleDebug

4) local.properties zip'te yok (SDK yolu senin PC'ye ozel).
   Android Studio ilk acilista SDK yolunu sorar / otomatik yazar.

5) APK genelde: app\build\outputs\apk\debug\app-debug.apk
